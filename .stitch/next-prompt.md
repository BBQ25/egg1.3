Create a desktop Stitch screen for the Device Registry page of the Real-Time Monitoring System of the Automated Poultry Egg Weighing and Sorting Device.

DESIGN SYSTEM (REQUIRED):
- Preserve Sneat admin patterns.
- Prefer compact desktop density.
- Use white or light neutral surfaces.
- No gradients.
- Use rounded cards, subtle shadows, and practical admin spacing.
- Use icon + label pairings for actions.
- Keep the screen operational and trustworthy rather than decorative.

Page Structure:
1. Top summary row with device health metrics
2. Search and action bar for device registration
3. Device table with status, owner, farm, last seen, and key actions
4. API key reveal flow area that feels secure and admin-only
5. Right-side or inline support panel for device key rules and ingest contract hints
