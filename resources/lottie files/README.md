# Curated LottieFiles Preview Shortlist

This folder contains a ranked shortlist of `42` `.mp4` preview files copied from the raw `downloads` dump for the egg monitoring system.

Source:
- `D:\Laragon Backup\downloader_helper\output\lottiefiles_illustrations_all_76098\downloads`

Why this exists:
- The raw library contains `51,383` preview files and is too large to browse manually.
- This shortlist prioritizes previews that best fit the poultry egg weighing, sorting, inventory, monitoring, and device-networking workflows of the system.

Ranking method:
- Higher rank means closer fit to the egg monitoring system domain.
- Ranking favors direct poultry and egg scenes first, then inventory/stock control, then monitoring dashboards, device/network visuals, and weighing/measurement visuals.

Folders:
- `01-poultry-egg-flow`: egg collection, hens, chickens, poultry farm scenes
- `02-inventory-stock`: inventory checks, stock handling, warehouse and item control
- `03-monitoring-dashboard`: monitoring, analytics, dashboard, and live-tracking visuals
- `04-devices-network`: IoT, connectivity, devices, sensors, and network visuals
- `05-weighing-measurement`: scales, weighing, and measurement-aligned visuals

Naming:
- Files are copied with an overall rank prefix, for example `R01__...mp4`
- The original raw filename is preserved after the rank prefix

Manifest:
- See `manifest.csv` for the ranked list, source filename, and fit rationale
